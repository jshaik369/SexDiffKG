# Abstract

**Background:** Drug safety signals differ between sexes, but no comprehensive organ-system-level mapping exists across the full pharmacopeia.

**Methods:** We analyzed 96,281 sex-differential drug safety signals from 14,536,008 FAERS reports (2004Q1-2025Q3) across 10 organ systems: cardiac, hepatic, renal, neurologic, psychiatric, respiratory, gastrointestinal, musculoskeletal, hematologic, and dermatologic. Each system was characterized by subcategory, drug class, and severity gradient.

**Results:** Organ systems span a 14-percentage-point range in female fraction: from hematologic (52.1%F, most male-enriched) to musculoskeletal (66.2%F, most female-enriched), against a FAERS baseline of 60.2%F.

Key findings across 10 organ systems:
- **Hematologic**: 3,122 signals, 52.1%F
- **Cardiotoxicity**: 3,792 signals, 53.6%F
- **Nephrotoxicity**: 2,382 signals, 54.9%F
- **Psychiatric**: 2,954 signals, 57.0%F
- **Respiratory**: 4,952 signals, 57.3%F
- **Metabolic Endocrine**: 3,993 signals, 60.2%F
- **Neurotoxicity**: 6,202 signals, 61.4%F
- **Gastrointestinal**: 4,476 signals, 61.8%F
- **Dermatologic**: 3,810 signals, 62.6%F
- **Musculoskeletal**: 3,218 signals, 66.2%F

Male-enriched systems (below 60.2%F baseline): hematologic (VTE 42.6%F), cardiac, nephrotoxicity, psychiatric, respiratory
Female-enriched systems (above baseline): musculoskeletal, dermatologic, gastrointestinal, metabolic/endocrine

Cross-cutting patterns:
1. **ICI Consistency**: Immune checkpoint inhibitors show male-biased signals across ALL organ systems (39.9-50.8%F), representing the most consistently male-biased drug class
2. **Anti-TNF Consistency**: Anti-TNF biologics show female-biased signals across ALL organ systems (63.3-86.3%F), the most consistently female-biased class
3. **Severity Gradient**: Serious AEs are more male-biased than non-serious AEs across all systems tested
4. **Drug Class Dominance**: Drug class is a stronger predictor of sex profile than organ system

**Conclusions:** Sex-differential drug safety signals form a coherent organ-system landscape where hematologic/cardiac/renal events are male-enriched and musculoskeletal/dermatologic/GI events are female-enriched. Drug class effects cross-cut organ system boundaries.

**Keywords:** sex differences, organ systems, pharmacovigilance, FAERS, drug safety, comprehensive mapping

# 1. Introduction

Sex differences in drug adverse effects are well-documented for individual drug-AE pairs but have never been systematically mapped across organ systems at population scale. Understanding the organ-system-level architecture of sex-differential signals is critical for:
- Clinical practice: sex-stratified monitoring protocols
- Drug development: anticipating sex-specific toxicity profiles
- Regulatory science: sex as a biological variable in safety evaluation

We present the first comprehensive organ-system mapping using 96,281 sex-differential signals from 14.5M FAERS reports.

# 2. Methods

## 2.1 Data Source
FAERS reports (2004Q1-2025Q3), 14,536,008 deduplicated (8.7M female, 5.8M male). Drug names normalized via DiAna dictionary. Sex-differential signals computed as female fraction of reporting.

## 2.2 Organ System Classification
Ten organ systems were defined using MedDRA preferred terms:
- **Cardiac**: QT prolongation, heart failure, MI, arrhythmias (3,792 signals)
- **Hepatic**: DILI, transaminase elevation, hepatitis (3,090 signals)
- **Renal**: AKI, CKD, proteinuria, dialysis (2,382 signals)
- **Neurologic**: Seizures, neuropathy, headache, CVA (6,202 signals)
- **Psychiatric**: Depression, anxiety, psychosis, suicidality (2,954 signals)
- **Respiratory**: Pneumonia, ILD, dyspnoea, PE (4,952 signals)
- **Gastrointestinal**: Nausea, diarrhoea, GI bleeding, colitis (4,476 signals)
- **Musculoskeletal**: Arthralgia, myopathy, fracture, tendon (3,218 signals)
- **Hematologic**: Cytopenias, VTE, bleeding, coagulopathy (3,122 signals)
- **Dermatologic**: Rash, alopecia, SJS/TEN, skin neoplasm (3,810 signals)

## 2.3 Analysis
Each system was analyzed by:
1. Overall sex profile (%F reporting)
2. Subcategory breakdown (AE-type level)
3. Drug class comparison (6-8 classes per system)
4. Severity gradient (serious vs non-serious where available)
5. Volume-sex correlation (anti-regression testing)

## 3.2 Cross-System Drug Class Effects

### ICIs: Consistent Male Bias
Immune checkpoint inhibitors show male-biased signals in every organ system:
- Respiratory: 39.9%F (most male)
- Nephrotoxicity: 44.3%F
- Cardiac: ~46%F
- Musculoskeletal: 46.8%F
- GI: 47.0%F
- Hematologic: 48.8%F
- Metabolic: 49.3%F
- Neurologic: 50.8%F

### Anti-TNFs: Consistent Female Bias
- MSK: 86.3%F (most female)
- GI: 79.2%F
- Respiratory: 77.8%F
- Hematologic: 63.3%F

## 3.3 Severity Gradient
Across all systems tested, serious AEs show lower female fractions than non-serious:
- Renal: Serious 51.3%F vs Non-serious 55.2%F
- Hepatic: Similar gradient
- This represents a universal severity-sex gradient

## 3.4 Notable Subcategory Findings
- Pulmonary vascular (PE, pulmonary HTN): 44.3%F — most male respiratory subcategory
- VTE: 42.6%F — most male hematologic subcategory
- Eating disorders: 72.0%F — most female psychiatric subcategory
- Alopecia: 76.2%F — strongly female dermatologic AE
- Rhabdomyolysis: 46.6%F — male-biased within female-biased MSK system
- ICI adrenal insufficiency: 45.8%F vs ICI thyroid: 62.6%F — divergent endocrinopathy

# 4. Discussion

## 4.1 The Organ System Sex Landscape
We identify a coherent organ-system architecture where:
- **Male-enriched** (hematologic, cardiac, renal): Systems associated with acute, life-threatening events
- **Female-enriched** (MSK, dermatologic, GI): Systems associated with chronic, quality-of-life impacts

## 4.2 Drug Class as Universal Modifier
Drug class overrides organ system: ICIs are male-biased everywhere, anti-TNFs female-biased everywhere. This suggests drug mechanism, not organ vulnerability, is the primary determinant.

## 4.3 Clinical Implications
1. Sex-stratified monitoring should prioritize cardiac/hematologic for males, MSK/dermatologic for females
2. ICI monitoring should account for the 10-20pp male shift across all organ systems
3. Severity-adjusted sex profiles provide more accurate risk assessment

# 5. Limitations
- FAERS voluntary reporting bias
- No drug exposure denominators
- Confounding by indication
- MedDRA term granularity
- 10 organ systems do not cover all possible AE categories

# 6. Conclusions
This first comprehensive organ-system mapping of sex-differential drug safety reveals a coherent landscape from male-enriched hematologic (52.1%F) to female-enriched musculoskeletal (66.2%F). Drug class effects, particularly ICI male bias and anti-TNF female bias, cross-cut all organ systems. These findings provide a framework for sex-stratified safety monitoring.
